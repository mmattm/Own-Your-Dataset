# Own-Your-Dataset

## ECAL 2025

### 1. Installer Node.js

Téléchargement : https://nodejs.org
Installer la version LTS.

### 2. Ouvrir le terminal

Se placer dans le dossier du projet :
cd chemin/vers/projet

Installer les dépendances :
npm install

### 3. Lancer le serveur Vite

npm run dev
Accès local :
http://localhost:5173/

### 4. Accès depuis un mobile (même Wi-Fi)

Utiliser l’adresse Network affichée par Vite, par exemple :
http://192.168.1.42:5173/
